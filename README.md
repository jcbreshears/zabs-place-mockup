# ZABS Place — Home Page Concept

A custom-coded home-page mockup for [ZABS Place](https://zabsplace.org), a nonprofit thrift
boutique in Matthews, NC that trains young adults with special needs for real careers.

**Live preview:** https://jcbreshears.github.io/zabs-place-mockup/

Hand-built responsive HTML/CSS in the ZABS Place brand palette and type system — warm, playful,
and mission-forward, with a clear path through every audience (shop, donate, give, apply).

> Note: a few figures (impact stats), the partner logos, and the testimonial are placeholders
> pending real content from ZABS Place.

Built by [Justin Breshears](https://jcbreshears.github.io/).
